    Módulo de consulta de clima usando APIs externas (Open-Meteo e Geocoding). Contém funções para validar cidade, buscar coordenadas, obter clima atual, gerar descrições e ícones, alternar tema dia/noite e exibir mensagens de erro no DOM. Elementos DOM integrados: - Formulário: #weather-form - Input cidade: #city-input - Resultado: #result - Botão Home: #home-btn Exemplo de uso assíncrono:

    Example

    const cidade = "Rio de Janeiro";
try {
  validarCidade(cidade);
  const location = await getCoordinates(cidade);
  const weather = await getWeather(location.latitude, location.longitude);
  const descricao = getWeatherDescription(weather.weathercode);
  const icone = getWeatherIcon(weather.weathercode);
  console.log(`Clima em ${location.name}, ${location.country}: ${descricao}, ${weather.temperature}°C`);
} catch (error) {
  console.error(error.message);
}
